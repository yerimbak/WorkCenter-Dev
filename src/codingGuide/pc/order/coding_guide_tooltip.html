  • 툴팁(호버 / 위쪽 방향 / 블루)

    버튼을 클릭하면 전체 접수 정보를 확인 하실 수 있습니다.

  • 툴팁(호버 / 아래쪽 방향 / 블루)

    버튼을 클릭하면 전체 접수 정보를 확인 하실 수 있습니다.

  • 툴팁(호버 / 위쪽 방향 / 핑크)

    버튼을 클릭하면 전체 접수 정보를 확인 하실 수 있습니다.

  • 툴팁(호버 / 아래쪽 방향 / 핑크)

    버튼을 클릭하면 전체 접수 정보를 확인 하실 수 있습니다.

  • 툴팁(예외 - 큰사이즈)

제품 입력 방식 툴팁 팝업

제품 입력 방식이란?
LG Chem On은 주문 시 두 가지 방식의 주문 제품 입력 방식을 지원하고 있습니다.
생산 단위를 가리키는 제품 코드를 정확히 모를 경우 제품명만
선택하셔도 주문 진행이 가능합니다.
제품명 & 포장 조합 입력이란?
LG화학 제품 코드를 모르신다고요?
제품 메뉴에 등록된 제품명으로 주문 요청할 수 있습니다.
제품명을 선택 후 주문 수량을 입력하시면 주문 요청이 완료됩니다. 만약 원하시는 포장 유형이 있다면 추가로 입력하시면 됩니다.
제품 코드 입력이란?
제품명과 포장 유형을 조합한 LG화학의 제품 코드를
이미 알고 계신다면, 바로 제품 코드를 입력하여 주문 요청을 완료하실 수 있습니다. 제품코드를 선택 후 주문 수량을 입력하시면 주문 요청이 완료됩니다.
